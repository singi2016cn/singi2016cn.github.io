# 借贷记账法

## 六大会计要素

会计要素是指会计核算中所反映的经济内容的基本分类。根据《企业会计准则》，六大会计要素包括：资产、负债、所有者权益、收入、费用和利润。

1. 资产：指企业过去的交易或者事项形成的、由企业拥有或者控制的、预期能够带来经济利益的资源。
2. 负债：指企业过去的交易或者事项形成的、预期会导致经济利益流出企业的现时义务。
3. 所有者权益：企业资产扣除负债后，由所有者享有的剩余权益。
4. 收入：指企业在日常活动中形成的、会导致所有者权益增加、与所有者投入资本无关的经济利益的总流入。
5. 费用：指企业在日常活动中发生的、会导致所有者权益减少、与向所有者分配利润无关的经济利益的总流出。
6. 利润：企业在一定会计期间经营成果，包括收入减去费用的净额、直接计入当期利润的利得和损失等。

## 动静结合式

资产 = 负债 + 所有者权益
资产 = 负债 + 所有者投入 + 留存收益
资产 = 负债 + 所有者投入 + 收入 + 利得 - 费用 - 损失
资产 + 费用 + 损失 = 负债 + 所有者投入 + 收入 + 利得

> 费用、损失称为费用类；等式左边是资金的来源，右边是资金的用途。

> 任何经济业务，都不可能打破“动静结合式”的平衡！

## 借贷记账法

资产 + 费用 + 损失 = 负债 + 所有者投入 + 收入 + 利得
借增贷减           = 借减贷增

> 有借必有贷，借贷必相等

> 借贷是一种符号，没有实际的意义。

### 会计分录

先定科目，再定借贷。

1. 用银行存款1000万购买原材料
```
借：原材料 10000000
    贷：银行存款 10000000
```
2. 购买50万设备，支付银行存储20万，剩余款项后续支付
```
借：固定资产 500000
    贷：银行存款 200000
        应付账款 300000
```
3. 接受投资者投入的价值500万的无形资产
```
借：无形资产 5000000
    贷：实收资本 5000000
```
4. 以银行存款100万偿还短期借款
```
借：短期借款 1000000
    贷：银行存款 1000000
```
5. 支付本月办公电费1万元
```
借：管理费用 10000
    贷：银行存款 10000
```
6. 销售货物，取得银行存款1000万，该货物成本800万。
```
借：银行存款 10000000
    贷：主营业务收入 10000000
借：主营业务成本 8000000
    贷：库存商品 8000000
```

> 收支两条线，分开核算。

### 会计分录三要素

- 会计科目
- 借贷方向
- 金额
