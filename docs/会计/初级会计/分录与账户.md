# 分录与账户

会计核算的对象是资金运动

记账凭证包括：原始凭证、记账凭证

## 会计科目与会计账户

T型账户是简易的会计账簿，因为它只能反应一个会计科目的变化。

会计账户是根据会计科目设置的，具有一定格式和结构，用于分类核算会计要素增减变动情况及其结果的载体。

### 账户结构

1. 借贷记账法下，账户的左方记为借方，右方记为贷方。左借右贷。
2. 一方等级增加，另一方等级减少。一增一减。
3. 账户余额通常在增加发生额乙方。

## 会计基础

权责发生制：收入、费用、资产、负债的确认与计量均以发生为基础，不考虑是否收到或支付款项。

收付实现制：收入、费用、资产、负债的确认与计量均以收到或支付款项为基础，不考虑是否发生。
