# 两大会计等式

## 会计分类

1. 按照行业

- 工业会计
- 商业会计
- 农业会计
- 金融会计
- 保险会计
- 公共会计
- 事业单位会计

2. 按照会计主体

- 企业会计
- 政府会计
- 非营利组织会计

## 会计是做什么的(账务处理程序)

1. 原始凭证：经济业务发生的证明，也是会计做账的凭据。
2. 记账凭证
3. 会计账簿
4. 财务报告：会计账簿的汇总，包括资产负债表、利润表、现金流量表、所有者权益变动表、附注。

> 期末余额 = 初期余额 + 本期增加额 - 本期减少额

## 资产 = 负债 + 所有者权益

- 资产：企业拥有或者控制的资源，是企业过去的交易或者事项形成的，预期会给企业带来经济利益或者产生效益的经济资源。
- 负债：企业过去的交易或者事项形成的，预期会导致经济利益流出企业的经济资源。
- 所有者权益：企业所有者对企业的资产拥有所有权或者控制权，并享有分配剩余资产的权利。

> 资产和负债是客观存在的，所有者权益是计算出来的。所有者权益也叫股东权益、剩余权益、净资产。

> 资产和负债是按照流动性分类的。（1年或1个正常的生产周期），所有者权益是按照来源分类：所有者投入 + 后续经营（留存收益）

## 收入 - 费用 = 利润

- 收入：企业在日常活动中形成的经济利益的总流入，导致所有者权益增加。
- 费用：企业在日常活动中形成的经济利益的总流出，
- 利润：企业在日常活动中形成的经济利益的总流入减去总流出。

> 营业收入 - 营业费用 = 营业利润

> 营业利润 + 营业外收入（利得） - 营业外支出（损失） = 利润总额（会计利润）

> 利润总额 - 所得税费用 = 净利润

## 资产负债表与利润表的关系

- 资产负债表：期初资产 = 期初负债 + 期初所有者权益
- 利润表：期初所有者权益 + 本期净利润 = 期末所有者权益
