# 会计科目

会计核算对象是资金运动。

资金运动 -> 六大会计要素 -> 会计科目

资产 + （费用 + 损失）  = 负债 + 所有者权益 + （收入 + 利得）
资产类科目 + 成本类科目 = 负债类科目 + 所有者权益类科目 + 损益类科目

> 损益类 包括费用、 损失、收入、利得。

> 对于损益类的科目，不用区分收入类还是费用类，借方都是减少利润，贷方都是增加利润。

> 成本类要么是资产，要么是费用，都是借方增加，贷方减少。

账面余额 = 科目余额

账面价值 = 账面余额 - 备抵科目

账面净值 = 账面余额 - 累计折旧/累计摊销（仅限于固定资产/无形资产）

## 计量属性

计量属性|概念|主要应用
---|---|---
历史成本（实际成本）|取得资产或发生负债时实际支付的现金或现金等价物的金额|企业对会计要素的计量一般采用历史成本
重置成本（现行成本）|重置或重建类似资产所需的现行成本|盘盈物资的计量
可变现净值|账面价值减去预计处置费用|存货的后续计量，期末按成本与可变现净值孰低计量
公允价值|在有序交易中，市场参与者之间自愿进行资产出售或负债清偿的价格|交易性金融资产、投资性房地产
现值|未来现金流量按某一适当的现值折现率折算的现值|长期投资、长期负债

> 公允价值计量属性下，没有备抵科目
